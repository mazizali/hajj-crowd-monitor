# Hajj Crowd Monitor 🕋

Vision AI dashboard for real-time crowd density monitoring at Masjid al-Haram and Mount Arafat.

**Made by Mahmoud AbdelAziz** · DevisionX

## Features

- **Live monitoring** of multiple Hajj streams with real-time density metrics
- **Heatmap analyzer** with pixel-level crowd analysis on any frame
- Snapshot capture: pause any stream → instant deep analysis
- Time-series trends, alert system, region breakdown
- Procedurally-generated demo frames (no external dependencies)

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Recharts (charts)
- Lucide React (icons)

## Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

## Deploy to Vercel

1. Push this folder to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — just click Deploy
